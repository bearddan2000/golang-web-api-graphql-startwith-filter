# golang-web-api-graphql-startwith-filter

## Description
Using an Apollo server, built-in, query static
data of 10 countries with internet codes. A filter
is set on the `Abbr` object to only return values
that start with 'b'.

## Tech stack
- bash
- golang 1.13

## Docker stack
- ubuntu:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
